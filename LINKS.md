## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/adrenolitik/DiabetView.git
cd DiabetView

# Установить зависимости
npm install

# Запустить dev сервер
npm run dev

# Собрать production
npm run build

# Развернуть на Vercel
npx vercel
```
