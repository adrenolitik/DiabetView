# 🚀 Быстрое развертывание на Vercel

## ⚡ Самый простой способ (5 минут)

### Шаг 1: Откройте Vercel
Перейдите по ссылке: **https://vercel.com**

### Шаг 2: Войдите через GitHub
Нажмите кнопку **"Sign Up"** или **"Log In"** и выберите **"Continue with GitHub"**

### Шаг 3: Импортируйте проект
1. После входа нажмите **"Add New..."** → **"Project"**
2. Найдите репозиторий **`DiabetView`** в списке
3. Нажмите **"Import"** рядом с ним

### Шаг 4: Настройки (автоматические)
Vercel автоматически определит:
- ✅ Framework: **Vite**
- ✅ Build Command: `npm run build`
- ✅ Output Directory: `dist`

**Ничего менять не нужно!**

### Шаг 5: Развертывание
Нажмите большую синюю кнопку **"Deploy"**

⏱️ Ожидайте 1-2 минуты...

### Шаг 6: Готово! 🎉
После завершения вы получите ссылку вида:
```
https://diabet-view-XXXXX.vercel.app
```

**Скопируйте и поделитесь этой ссылкой!**

---

## 🔧 Опциональная настройка API

Если хотите использовать настоящий Google Gemini AI (не обязательно):

1. В настройках проекта на Vercel найдите **"Environment Variables"**
2. Добавьте переменную:
   - **Name**: `VITE_GOOGLE_API_KEY`
   - **Value**: ваш API ключ от Google
3. Нажмите **"Save"**
4. Vercel автоматически пересоберет проект

> **Примечание**: Без API ключа приложение работает с демо-данными. Это нормально!

## ❓ Возникли проблемы?

### Ошибка при сборке
Проверьте, что в репозитории есть все файлы (особенно `package.json` и `src/` директория)

### Страница не загружается
Подождите 1-2 минуты после деплоя. Vercel может кешировать результат.

### API не работает
Это нормально! Без API ключа приложение показывает демо-данные.
