# DiabetView: Counterfactual Health Visualizer 🩺

Интерактивный прототип медицинского приложения, демонстрирующий пациентам с диабетом сценарии "Что если..." с использованием генеративного ИИ.

## 🌟 Возможности

*   **Профиль пациента**: Ввод клинических показателей (HbA1c, вес, давление, стаж).
*   **AI Симуляция (Google Gemini)**: Генерация прогнозов риска и объяснений на естественном языке.
*   **Counterfactual Analysis**: Слайдеры интервенций (снижение веса, контроль глюкозы) показывают альтернативное будущее.
*   **Визуализация последствий**:
    *   *Лицо*: Симуляция внешних признаков здоровья vs болезни.
    *   *Зрение*: Эмуляция диабетической ретинопатии (размытие, скотомы).
    *   *Сосуды*: Визуализация атеросклероза и расчет рисков.
    *   *Долголетие*: Прогноз ожидаемой продолжительности жизни.

## 🛠 Технологический стек

*   **Frontend**: React 18, TypeScript, Vite
*   **AI**: Google Gemini API (Multimodal: Text & Vision)
*   **Стилизация**: Tailwind CSS
*   **Графики**: Recharts

## 🚀 Запуск локально

Для запуска вам понадобится Node.js 18+.

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/your-username/DiabetView.git
    cd DiabetView
    ```

2.  Установите зависимости:
    ```bash
    npm install
    ```

3.  Создайте файл `.env.local` в корне проекта и добавьте ваш API ключ:
    ```env
    VITE_GOOGLE_API_KEY=ваш_ключ_gemini_api
    ```
    *(Примечание: Приложение работает с mock-данными, если ключ не указан)*

4.  Запустите проект:
    ```bash
    npm run dev
    ```
    Откройте http://localhost:5173

5.  Создайте production сборку:
    ```bash
    npm run build
    npm run preview
    ```

## 🌐 Развертывание на Vercel

### Быстрый способ (через CLI):

1.  Установите Vercel CLI:
    ```bash
    npm install -g vercel
    ```

2.  Войдите в Vercel:
    ```bash
    vercel login
    ```

3.  Разверните проект:
    ```bash
    vercel
    ```

4.  Для production деплоя:
    ```bash
    vercel --prod
    ```

### Через веб-интерфейс:

1.  Перейдите на [vercel.com](https://vercel.com)
2.  Нажмите "Import Project"
3.  Выберите ваш Git репозиторий
4.  Vercel автоматически определит настройки Vite
5.  (Опционально) Добавьте переменную окружения `VITE_GOOGLE_API_KEY` в настройках проекта
6.  Нажмите "Deploy"

### Конфигурация переменных окружения:

В настройках Vercel проекта добавьте:
- `VITE_GOOGLE_API_KEY` - ваш ключ Google Gemini API

## ⚠️ Дисклеймер

Это приложение является MVP (Minimum Viable Product) и демонстрацией технологий. Оно **не является медицинским устройством**. Все прогнозы и визуализации носят иллюстративный характер.
